SANVIA - 情緒虛擬商品平台

探索全端開發流程，將抽象的情緒價值具體化為虛擬商品

SANVIA

  • 專案簡介：結合 VIA (通過) 跟 SAN 值 的概念，將抽象的情緒價值具體化為虛擬商品，為期一個月的獨立開發結訓專題，旨在探索全端開發流程。
  • 技術棧：Vue3 + Pinia + Node.js + Express.js + MongoDB
  • 專案現況：目前已完成商品瀏覽、購物車、會員 JWT 註冊登入、RESTful API 串接與資料庫管理等功能，未來期許擴充社群互動模組，讓平台不只提供交易，更建立社交與情緒價值連結。
  • 備註：因後端平時處於休眠狀態，需手動刷新並耐心等候。

文輕小站

職場 21 點 - 翻牌點擊遊戲

結合經典撲克牌遊戲 21 點 (Blackjack) 與趣味點擊打 Boss 小遊戲

職場 21 點

  • 專案簡介：這是一個結合經典撲克牌遊戲 21 點 (Blackjack) 與趣味點擊打 Boss 小遊戲的電腦尺寸網頁遊戲。在忙碌的辦公室場景中，挑戰你的運氣和反應力，為社畜的職場生存而戰！
  • 技術棧：jQuery + HTML + CSS + JS (Vanilla JS)
  • 專案現況：完成核心邏輯 21 點遊戲規則（含 A 點數彈性及洗牌），利用 CSS 3D 翻轉動畫及 JS 異步控制發牌時序，並利用 localStorage 儲存打 Boss 小遊戲的最高分數。
  • 備註：僅適合電腦尺寸，無 RWD 。

可愛數字鐘 - JS 練習

能夠即時且準確地顯示時間和日期，純前端展示應用

可愛數字鐘

  • 專案簡介：簡潔、可愛風格的介面，用來顯示即時的當前時間與日期。
  • 技術棧：HTML + CSS + JavaScript (Vanilla JS) + Particles.js
  • 專案現況：已完成自動更新時間，小時與分鐘數字在變化時具有不同動畫效果。每秒跳動彩虹點；分鐘個位數：圓環排列數字；分鐘十位數 / 小時十位數：翻牌動畫；小時個位數：垂直滾動動畫。
  • 備註：僅適合電腦尺寸，無 RWD 。

顏色選擇器 - JS 練習

可以透過 RGB 滑桿或直接輸入 Hex Code 來調整顏色

顏色選擇器

  • 專案簡介：一個現代、功能豐富且互動性極高的 RGB/Hex 雙向顏色選擇器。 這個專案旨在提供一個直覺的介面，讓使用者可以透過 RGB 滑桿或直接輸入 Hex Code 來調整顏色，並即時看到背景顏色的變化。

基於課堂範例，加上切換主題模式的功能

番茄鐘
